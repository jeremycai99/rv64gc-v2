# Documentation Archive

This directory preserves dated investigations, superseded plans, and provenance
material that should not crowd the active `doc/` index.

## Directories

- `4wide/` - pivot, refactor, signoff, and gap-closure iteration history.
- `reference_core/` - earlier competitor-core analysis and feature-matrix
  notes superseded by `../reference_core_unified_audit_2026-05-03.md`.
- `debug_history/` - old CoreMark, Dhrystone, dsim, and ASIC signoff handoffs.
- `design_experiments/` - shelved or deferred design experiments.

Archived docs are kept for traceability. New decisions should usually update a
top-level current doc instead of appending to archived provenance.
